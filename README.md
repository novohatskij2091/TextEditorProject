# Языковой процессор - Текстовый редактор

## 📋 Описание
Текстовый редактор с графическим интерфейсом, разработанный для лабораторной работы №1.
В дальнейшем будет дополнен функциями языкового процессора.

## ✨ Возможности
- Создание новых текстовых файлов
- Открытие существующих файлов (.txt)
- Сохранение файлов
- Редактирование текста (копировать, вставить, вырезать)
- Отмена и повтор действий
- Анализ текста (подсчет строк и символов)
- Интуитивно понятный интерфейс

## 🚀 Установка и запуск

### Вариант 1: Готовая программа (проще всего)
1. Скачайте файл `LanguageProcessor.exe`
2. Запустите его двойным щелчком мыши
3. Ничего устанавливать не нужно!
